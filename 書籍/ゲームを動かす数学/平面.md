# 3次元

## 平面

平面を構成するための条件は二つある
- 同一直線上にない座標が3点決まっている
- 平面の法線と平面上の座標1点が決まっている

## 平面の方程式

`a(X - Xo) + b(Y - Yo) + C(Z - Zo) = 0`  
(a,b,c) : 平面上の法線  
(Xo,Yo,Zo) :平面上の座標

この式が成り立つ(x,y,z)は`平面上の座標`である。  
また、結果が`正`なら`表面側に存在し`、`負`なら`裏面に存在する`


この式に置き換えることも可能  
`aX + bY + cZ + d = 0`  
dは`aXo + bYo + cZo + d = 0`で求まる。
